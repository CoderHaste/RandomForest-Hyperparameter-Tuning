# ⚙️ Random Forest Hyperparameter Tuning using RandomizedSearchCV

## 📌 Overview

This project focuses on improving machine learning model performance using hyperparameter tuning techniques.
RandomizedSearchCV is used to optimize a Random Forest classifier and improve prediction accuracy on a customer churn dataset.

The project demonstrates how tuning parameters and cross-validation can significantly enhance model performance.

---

## 🚀 Features

* Data preprocessing and cleaning
* Label encoding for categorical variables
* Feature scaling
* Initial Random Forest model training
* Hyperparameter tuning using RandomizedSearchCV
* Model evaluation and comparison
* Cross-validation for robustness

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## 📊 Techniques Used

* Random Forest Classifier
* RandomizedSearchCV
* Cross-validation
* Model evaluation using accuracy & classification report
